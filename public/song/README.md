# Songs Folder

Place your **song1.mp4a** (or .mp3) here.

This is the background music that auto-plays at 40% volume when visitors load your portfolio.

## Instructions:
1. Add your music file to this folder
2. Name it `song1.mp4a` or `song1.mp3`
3. The path should be: `/public/song/song1.mp4a`
4. The application will automatically play it (if browser allows autoplay)
5. Users can control playback with the floating music button in the bottom-right corner

## Supported Formats:
- MP3 (recommended)
- MP4A
- OGG
- WAV

**Tip:** Keep file size under 10MB for fast loading!
